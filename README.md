# Data Curation Environment (DCE) for VS Code

The Data Curation Environment is a VS Code extension designed to streamline AI-assisted development workflows. It provides a suite of tools to help developers and researchers curate, manage, and package context for Large Language Models (LLMs).

## Core Features

*   **Context Chooser:** An intuitive file tree with checkboxes to easily select files and folders for your AI prompt's context.
*   **Parallel Co-Pilot Panel:** A multi-tabbed environment to manage, compare, and test multiple AI-generated responses side-by-side.
*   **Integrated Diff Viewer:** Quickly see the differences between your workspace files and the AI's suggestions.
*   **Cycle-Based History:** A persistent, navigable history of your interactions, creating a knowledge graph of your project's evolution.
*   **Git Integration:** Create baseline commits and restore your workspace with one-click actions, enabling a rapid and safe test-and-revert loop.
*   **Onboarding:** A "Cycle 0" experience to help new users bootstrap their projects by generating an initial set of planning and documentation artifacts.
